Skullmonkeys — sprite naming review

Each tile is an uncracked sprite, captioned with its 32-bit id, level-count (L#), and a known role or the engine function that references it. Recognise a sprite? Note its 0x… id + your guessed name and send it back — I hash-check each with calcHash (RAW = id, or WRAP = 0x28C0E011^rotl(calcHash,27)). A name is only trusted if the picture fits and a sibling id corroborates.

Sheet 0 is the highest-value set (known bosses/items/text first); later sheets are ordered by level. Tap a sheet to open it full-size for pinch-zoom.
