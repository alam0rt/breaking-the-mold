# Container-confined sister families

_Generated by `tools/scripts/container_sisters.py` (thr=3, min=3)._

A **family** = asset IDs that ship in the same BLB container and whose hashes differ by ≤3 toggled bits (the `STEM + <short token>` signature). Container confinement removes cross-level collision noise. One confirmed member predicts the rest via `tools/scripts/family_structure.py`.

- per-container family instances (≥3 members): **175**  (sprite + audio)
- **distinct super-families** (instances merged across containers): **25** (see `super_families.csv`)
- super-families with an anchor: **13**  /  new: **12**
- families carrying a known/visual anchor (per-instance): **44**
- alias groups (distinct IDs, byte-identical payload): **26** (see `container_aliases.csv`)
- enumerated runs (IDs sharing top-20 bits = frame/state sequence): **5** (see `enumerated_runs.csv`)
